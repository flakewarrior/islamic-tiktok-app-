# Islamic TikTok App

This project automates posting Islamic educational content (quizzes, duas, reminders, lectures) to TikTok using **Airtable** and **n8n**.  
It is designed to help creators maintain a consistent, balanced feed without manual effort, while ensuring compliance with TikTok’s Developer requirements.

## Features
- Automated video posting via TikTok Content Posting API
- Airtable integration for managing content
- OAuth authentication for secure API access
- GitHub Pages hosting for Terms of Service and Privacy Policy

## Setup
1. Clone this repository.
2. Add your `terms.html` and `privacy.html` files to the root folder.
3. Configure GitHub Pages under **Settings → Pages** (branch: `main`, folder: `/root`).
4. Deploy and use the generated URLs in the TikTok Developer Portal.
